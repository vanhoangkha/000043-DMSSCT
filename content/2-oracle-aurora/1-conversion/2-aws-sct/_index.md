#### Install the AWS Schema Conversion Tool (AWS SCT)

Now that you are connected to the EC2 instance, you are going to install the AWS Schema Conversion tool on the server. Downloading the file and installing it will give you the latest version of the AWS Schema Conversion Tool.

1. On the **EC2 server**, open the **DMS Workshop** folder that is on the **Desktop**. Then, double-click on **AWS Schema Conversion Tool Download** to get the latest version of the software.

![Conversion](/images/2/6a.png?width=90pc)

2. When the download is complete, **unzip** the content and **install** the AWS Schema Conversion Tool.

{{% notice note %}}
When the installer is complete the installation dialog will disappear. There is no other notification.
{{% /notice %}}

3. Once the installation is complete, go to the **Start Menu** and **launch** the AWS Schema Conversion Tool.

![Conversion](/images/2/6b.png?width=90pc)

4. **Accept** the terms and Conditions.

![Conversion](/images/2/7.png?width=90pc)
