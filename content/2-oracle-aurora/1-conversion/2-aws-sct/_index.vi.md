#### Cài đặt AWS Schema Conversion Tool (AWS SCT)

Bây giờ bạn đã kết nối với máy ảo EC2, bạn sẽ cài đặt công cụ AWS Schema Conversion trên máy. Tải xuống tập tin và cài đặt nó sẽ cung cấp cho bạn phiên bản mới nhất của công cụ AWS Schema Conversion.

1. Trên **máy chủ EC2**, mở thư mục **DMS Workshop** trên **Desktop**. Sau đó, mở trang **AWS Schema Conversion Tool Download** để tải phiên bản mới nhất của phần mềm.

![Conversion](/images/2/6a.png?width=90pc)

2. Sau khi tải xong, **giải nén** nội dung và **cài đặt** AWS Schema Conversion Tool.

{{% notice note %}}
Sau khi cài đặt hoàn thành, cửa sổ cài đặt tắt đi. Sẽ không có thông báo nào khác được hiển thị.
{{% /notice %}}

3. Sau khi quá trình cài đặt hoàn thành, đi đến **Start Menu** và **chạy** ứng dụng AWS Schema Conversion Tool.

![Conversion](/images/2/6b.png?width=90pc)

4. **Chấp nhận** các Điều khoản sử dụng.

![Conversion](/images/2/7.png?width=90pc)
