# Di chuyển Database trên AWS

In this workshop, we will explore how you can migrate your data to and from most widely used commercial and open-source databases using the [**AWS Database Migration Service (DMS)**](https://aws.amazon.com/dms/) and [**AWS Schema Conversion Tool (AWS SCT)**](https://aws.amazon.com/dms/schema-conversion-tool/?nc=sn&loc=2).
